    Manual: copy & paste the script yourself · fallback

    Use this if Playwright isn't installed or you'd rather drive your own browser.

    1. Open the creator's profile in your browser, e.g. https://www.douyin.com/user/MS4wLjABAAAA….
    2. Open DevTools (F12) and switch to the Console tab.
    3. Click Open in new tab then paste it into the console and press Enter.
    4. The script walks every page of videos and downloads a douyin-video-data-…json file. Wait for the alert.
    5. Drop that JSON file into the box below.
